# android-eventos-java

Ejemplos básicos de gestión del evento onClick en Android con Java.

El proyecto tienes varias ramas (branches) para poder comparar los distintos modos de hacerlo:

- [[main] Versión con 3 variantes](https://github.com/avidaldo/android-eventos-java/tree/main)
- [[receptor_eventos] Variante con Receptor de eventos: sencilla pero no recomendada](https://github.com/avidaldo/android-eventos-java/tree/receptor_eventos)
- [[custom-OnClickListener] Variante creando una clase con un listener personalizado](https://github.com/avidaldo/android-eventos-java/tree/custom-OnClickListener)
- [[activity_como_listener] Variante implementando el listener en la propia Activity](https://github.com/avidaldo/android-eventos-java/tree/activity_como_listener)
- [[listener-clase-anonima] Variante utilizando una clase anónima](https://github.com/avidaldo/android-eventos-java/tree/listener-clase-anonima)
- [[pasando-funcion-lambda] Variante pasando función lambda](https://github.com/avidaldo/android-eventos-java/tree/pasando-funcion-lambda) (directamente equivalente a la anterior, el IDE convierte entre ellas automáticamente)
  
  
Para moverse entre ramas con git, basta hacer "git switch nombre-de-rama" desde el terminal de git en el directorio del repositorio, una vez este ha sido clonado.
  
  
  
[Documentación sobre eventos](https://developer.android.com/guide/topics/ui/ui-events#java)
